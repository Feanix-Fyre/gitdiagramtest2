# Tooltip

The `Tooltip` class represents a tooltip that can be attached to a UI element.

## Public Properties

| Property      | Type    | Description                                      |
| ------------- | ------- | ------------------------------------------------ |
| `text`        | `string`| The text to display in the tooltip.              |
| `labelOffset` | `Vector2` | The offset of the tooltip's label.               |
| `labelPosition` | `Vector3` | The position of the tooltip's label.             |
| `isWorldspace`| `bool`  | A boolean that indicates whether the tooltip is in world space. |
