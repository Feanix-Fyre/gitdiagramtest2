# AudioSettings

The `AudioSettings` class is a serializable class that contains settings for audio.

## Properties

| Name | Type | Description |
| --- | --- | --- |
| `MasterVolume` | `float` | The master volume. |
| `AmbientVolume` | `float` | The ambient volume. |
| `MusicVolume` | `float` | The music volume. |
| `SFXVolume` | `float` | The sound effects volume. |
| `UIVolume` | `float` | The UI volume. |
| `DialogueVolume` | `float` | The dialogue volume. |
| `FootstepsVolume` | `float` | The footsteps volume. |
