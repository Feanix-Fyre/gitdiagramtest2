# ToggleableLight

The `ToggleableLight` class represents a light that can be toggled on and off.

## Properties

| Name | Type | Description |
| --- | --- | --- |
| `isOn` | `bool` | If `true`, the light is on. |
| `MaterialIndex` | `int` | The material index for the light surface. |

## Methods

### `TurnOn()`

Turns the light on.

### `TurnOff()`

Turns the light off.
